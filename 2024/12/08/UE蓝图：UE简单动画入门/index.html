从一个简单的动画学习

UE GameMode

在虚幻引擎（Unreal Engine，简称UE）中，GameMode 是一个非常重要的类，它定义了游戏的基本规则和行为。GameMode 控制游戏的流程，如玩家如何加入游戏、游戏结束条件、玩家角色的生成等。它通常用于定义游戏的核心逻辑和规则，包括：

  1. 玩家角色的生成：决定玩家在游戏开始时使用哪个角色类（Player Character）。
  2. 玩家控制器：定义玩家控制角色的方式，如输入如何转换为角色的动作。
  3. HUD（Heads-Up Display）：管理游戏的用户界面，如得分、健康值等。
  4. 游戏状态（Game State）：用于跟踪游戏的全局状态，如得分、时间、玩家状态等。

GameMode 类常用于多人游戏和单人游戏的不同规则设定，可以通过继承和扩展这个类来自定义游戏的特定行为。例如，可以创建一个专门用于团队死亡竞赛模式的 TeamDeathmatchGameMode 类，或者一个用于竞速游戏的 RaceGameMode 类。

打开现有的 GameMode

  1. 在编辑器中打开项目：启动 Unreal Engine 并打开您的项目。
  2. 查找现有的 GameMode：
    • 在内容浏览器中，导航到包含 GameMode 类的文件夹（通常在 Blueprints 或 Gameplay 文件夹中）。
    • 搜索或直接查找 .uasset 文件，这些文件通常是蓝图类型的 GameMode 实例。
  3. 双击打开：双击 GameMode 蓝图文件即可打开它的编辑器窗口。

创建新的 GameMode

  1. 创建新的蓝图类：
    • 右键单击内容浏览器的空白区域，选择 Blueprint Class。
    • 在弹出的窗口中，选择 GameModeBase 或 GameMode 作为父类。GameModeBase 是一个更轻量的基类，而 GameMode 是一个更完整的功能类。
  2. 命名和保存：
    • 为新的 GameMode 类命名，并选择一个保存路径。
  3. 编辑新建的 GameMode：
    • 双击新建的 GameMode 蓝图类，进入蓝图编辑器。
    • 在这里，您可以设置默认的玩家Pawn(也可以选择新建一个character参考后面的步骤)、玩家控制器类、HUD 类等。

替换项目中的默认 GameMode

  1. 打开项目设置：
    • 在编辑器顶部的菜单栏中，选择 Edit > Project Settings。
  2. 导航到地图和模式设置：
    • 在项目设置窗口中，导航到 Maps & Modes 部分。
  3. 设置默认 GameMode：
    • 在 Default Modes 部分，找到 Default GameMode 设置。
    • 点击下拉菜单，然后选择您刚刚创建的 GameMode 类。
  4. 保存设置：
    • 确保所有更改都已保存。

完成这些步骤后，项目将使用新的 GameMode 作为默认的游戏模式。还可以通过编辑 GameMode 来进一步定制游戏的行为和规则。
